# 宝塔面板API配置指南

## 当前状态
✅ Node.js应用已成功部署到阿里云服务器  
✅ 应用在3001端口正常运行  
✅ MySQL数据库在3307端口正常运行  
❌ 需要配置宝塔面板网站来访问API服务  

## 配置步骤

### 1. 登录宝塔面板
- 访问: http://47.115.230.178:8888
- 使用您的宝塔面板账号密码登录

### 2. 添加网站
1. 点击左侧菜单「网站」
2. 点击「添加站点」
3. 填写配置:
   - **域名**: `47.115.230.178:3001` 或您的域名
   - **根目录**: `/www/wwwroot/shuijing-erp`
   - **PHP版本**: 选择「纯静态」
   - **数据库**: 不创建

### 3. 配置反向代理
1. 在网站列表中找到刚创建的网站
2. 点击「设置」
3. 选择「反向代理」标签
4. 点击「添加反向代理」
5. 填写配置:
   - **代理名称**: `shuijing-erp-api`
   - **目标URL**: `http://127.0.0.1:3001`
   - **发送域名**: `$host`
   - **内容替换**: 留空

### 4. 配置防火墙
1. 点击左侧菜单「安全」
2. 在「防火墙」中添加端口规则:
   - **端口**: `3001`
   - **协议**: `TCP`
   - **策略**: `放行`
   - **备注**: `水晶ERP API服务`

### 5. 配置SSL（可选）
如果需要HTTPS访问:
1. 在网站设置中选择「SSL」
2. 申请Let's Encrypt免费证书
3. 强制HTTPS

## 故障排除

### 502 Bad Gateway错误
- 检查Node.js应用是否在3001端口运行
- 检查反向代理配置是否正确
- 检查防火墙是否开放3001端口

### 连接超时
- 检查阿里云安全组是否开放3001端口
- 检查服务器防火墙设置
- 确认应用正在监听正确的端口

### 数据库连接错误
- 检查MySQL是否在3307端口运行
- 验证数据库用户权限
- 检查.env文件配置
